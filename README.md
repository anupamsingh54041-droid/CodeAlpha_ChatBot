
# 🤖 AI ChatBot using Java

An interactive console-based AI ChatBot developed using Java for the CodeAlpha Java Programming Internship.

This project simulates human-like conversation using rule-based logic and basic Natural Language Processing (NLP) techniques. The chatbot can respond to greetings, coding-related questions, internship-related queries, and general conversational inputs.

---

## 📌 Project Overview

The AI ChatBot is designed to interact with users in real-time through a console interface. It accepts user input, analyzes keywords, and generates appropriate responses based on predefined rules and conditions.

The main objective of this project is to demonstrate the implementation of Java programming fundamentals, string handling, loops, conditional logic, and basic NLP-like keyword matching techniques.

## 🧠 AI & NLP Concepts Used

This chatbot uses simple rule-based artificial intelligence techniques along with basic Natural Language Processing (NLP)-like behavior.

### Implemented Concepts:
- Keyword matching using `contains()`
- Case-insensitive input handling using `toLowerCase()`
- User intent recognition
- Pattern-based response generation
- Interactive conversational flow

---

## ⚙ How the ChatBot Works

1. The chatbot continuously waits for user input.
2. User messages are converted into lowercase format.
3. The chatbot checks for keywords using conditional statements.
4. Matching responses are displayed instantly.
5. The conversation continues until the user types:
   - `bye`
   - `exit`
   - `goodbye`

---

## ▶️ How to Run the Project

1. Open the project in any Java IDE
2. Run the `Main.java` file
3. Start chatting with the bot through the console
4. Type `bye` to end the conversation
